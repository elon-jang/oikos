# 코딩 몰라도 괜찮아요 - Claude Code로 업무 자동화한 이야기

안녕하세요. 오늘은 제가 Claude Code를 활용해서 교회 기부금 영수증 발행 업무를 자동화한 경험을 나눠볼게요. 코딩을 모르시는 분들께 도움이 되었으면 합니다.

---

## 이런 고민, 혹시 있으신가요?

매년 반복되는 업무가 있어요. 교회에서 94명 교인분들께 기부금 영수증을 발행하는 일인데요. 엑셀에서 이름과 금액을 확인하고, Word 양식에 하나씩 옮겨 적고, 저장하고... 이걸 94번 반복하면 반나절이 훌쩍 지나갑니다.

"이거 자동으로 안 될까?" 싶었는데, 프로그램을 만들자니 코딩을 모르겠고. 외주를 맡기자니 비용이 부담되고. 그러다 Claude Code를 알게 됐어요.

---

## Claude Code는 이렇게 써요

특별한 게 아니에요. 그냥 **하고 싶은 일을 말로 설명하면 됩니다**.

저는 이렇게 말했어요.

> "엑셀 파일에 이름이랑 월별 헌금 금액이 있어. 이걸 읽어서 Word 영수증 양식에 자동으로 채워주는 프로그램 만들어줘."

그랬더니 정말 만들어주더라고요. 처음 결과물이 마음에 안 들면 다시 말하면 돼요.

> "영수증이 A4 한 장에 안 들어가. 템플릿을 내가 만들 테니까, 거기에 값만 넣어줘."

이런 식으로 대화하듯 주고받으면서 점점 원하는 모습이 되어갔어요.

---

## 이런 것도 되더라고요

처음엔 기본 기능만 있었는데, 쓰다 보니 이것저것 필요해졌어요. 그때마다 Claude에게 부탁했습니다.

- **"부부는 이름이 같이 적혀있는데, 각각 따로 만들어줘"** → 쉼표로 구분된 이름을 자동으로 나눠서 처리
- **"특정 사람 것만 다시 발행하고 싶어"** → 이름을 지정하면 그 사람 것만 생성
- **"누구한테 언제 발행했는지 기록 남겨줘"** → 발행대장 엑셀 파일이 자동 생성
- **"내년에도 쓸 건데 연도를 매번 바꾸기 귀찮아"** → 파일명에서 연도를 자동으로 읽어서 처리

제가 한 건 "이렇게 해줘"라고 말한 것뿐이에요. 기술적인 부분은 Claude가 알아서 해결해줬습니다.

---

## 결과는요

**전**: 엑셀 보면서 Word에 복사-붙여넣기, 94번 반복, 3시간 이상

**후**: 명령어 한 줄 실행, 3초

남는 시간에 영수증 내용이 맞는지 꼼꼼히 확인할 수 있게 됐어요. 예전엔 만드느라 바빠서 검토할 여유가 없었거든요.

---

## 비개발자분들께 드리는 팁

**1. 원하는 걸 구체적으로 말해보세요**

"자동화해줘"보다는 "엑셀의 A열 이름을 Word의 이 위치에 넣어줘"처럼 구체적으로 말하면 결과가 더 정확해요.

**2. 한 번에 완벽하지 않아도 괜찮아요**

처음 결과물이 마음에 안 들면 "이 부분은 이렇게 바꿔줘"라고 하면 됩니다. 대화하면서 다듬어가는 거예요.

**3. 샘플 파일이 있으면 좋아요**

실제 데이터 구조를 보여주면 Claude가 더 정확하게 이해해요. 개인정보는 가명으로 바꿔서 보여주시면 됩니다.

## 마치며

코딩을 몰라도 **"무엇이 필요한지 설명하는 것"**은 누구나 할 수 있어요. 매일, 매주, 매년 반복하는 일이 있다면 한번 Claude Code에게 물어보세요.

"이거 자동으로 할 수 있을까?"

생각보다 많은 것들이 가능하더라고요.

혹시 궁금한 점 있으시면 편하게 물어봐 주세요. 😊
